# 🚀 Featured Projects

## 🌦 ESP32 IoT Weather Station

A smart weather monitoring system built using:

- ESP32
- DHT11 Temperature & Humidity Sensor
- Open-Meteo API
- Wi-Fi Communication
- HTTP Requests
- JSON Parsing

### Features

- Real-time temperature monitoring
- Real-time humidity monitoring
- Live weather updates from Open-Meteo
- Comparison of sensor data with online weather data
- Responsive web dashboard (under development)
